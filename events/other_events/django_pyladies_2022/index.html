    Do you want to learn how to code your own web application? In this workshop for non-experienced women and LGBTIQ+ people, organised by PyLadiesBCN, we’ll learn how to code our own blog with Django using HTML, CSS and Python. Throughout the whole day we’ll learn how web technology works, with the help of mentors that will guide us during the workshop. No previous knowledge required. Just bring your own laptop and be ready to learn and have fun!



    Previous Next

    What will it look like?

    Hands-on Django with PyLadiesBCN is a one-day hands-on workshop focused on introducing 32 attendees* with no previous experience in the web development world. Led by professional web development women and working in small groups (4 attendees + 1 coach), attendees will learn how to create and deploy their first blog application using HTML, CSS, Python and the Django framework.

    We will also host a series of 5’ lightning talks where different women working in tech will share their path and their experience with the attendees.

    There will be free lunch and coffee breaks provided during the day for attendees and volunteers. This is a non-profit event, free for all participants. Speakers, coaches and organisers are volunteers.

    We will host a Kahoot game with questions about PyLadiesBCN, women in tech and Python, with prizes for the fastest players. Will you miss it?!

    * We’ll prioritise women and members of the LGBTIQ+ community.
